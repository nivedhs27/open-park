# 🚗 Open Park – Parking Space Detection System

Open Park is a smart parking monitoring system developed to detect available parking spaces using computer vision techniques. The project uses OpenCV for image processing and video stabilization to improve detection accuracy.

---

## 📌 Overview

In crowded areas, finding parking space is often time-consuming. Open Park aims to simplify this by analyzing video input and identifying free parking slots, making parking more efficient and organized.

---

## ⚙️ Features

* Detects available parking spaces from video input
* Uses computer vision techniques for analysis
* Implements video stabilization for improved detection accuracy
* Practical implementation of real-world problem solving

---
